# Key Risk Indicators (KRIs)

## What is a KRI?

A KRI is a measurable value that indicates the level of risk exposure.

## Submitting KRI Values

1. Navigate to **KRIs** in the sidebar
2. Find your assigned KRI
3. Click to open details
4. Enter the current value and submit

## Understanding Limits

- **Green** — Within acceptable limits
- **Amber** — Approaching limits (warning)
- **Red** — Outside limits (breach)
