# Utility Workflows

## Research & Enhancement
- `/web-search` — Online research, synthesize findings
- `/prompt-enhancer` — Improve prompts for AI agents

## Skills Management
- `/skill-creator` — Create/update specialized skills
- `/use-skills` — Discover and invoke available skills

## Creative
- `/theme-factory` — Apply themes to artifacts
- `/slack-gif-creator` — Animated GIFs for Slack
- `/image-enhancer` — Improve screenshot quality
- `/video-downloader` — Download videos for offline use

## Workflow Selection

| Task | Workflow |
|------|----------|
| Research topic | `/web-search` |
| Better prompt | `/prompt-enhancer` |
| New skill | `/skill-creator` |
| Apply theme | `/theme-factory` |
| Slack GIF | `/slack-gif-creator` |
| Enhance image | `/image-enhancer` |
