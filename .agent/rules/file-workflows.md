# File & Document Workflows

## Office Documents
- `/pdf` — Extract, create, merge, split, fill forms
- `/docx` — Word docs with tracked changes, comments
- `/xlsx` — Spreadsheets with formulas, formatting
- `/pptx` — Presentations with layouts, speaker notes

## Documentation
- `/doc-coauthoring` — Proposals, specs, structured content
- `/changelog-generator` — Release notes from git commits
- `/internal-comms` — Status reports, newsletters, FAQs

## Workflow Selection

| Task | Workflow |
|------|----------|
| Fill PDF form | `/pdf` |
| Edit Word doc | `/docx` |
| Analyze spreadsheet | `/xlsx` |
| Create presentation | `/pptx` |
| Write proposal | `/doc-coauthoring` |
| Generate changelog | `/changelog-generator` |
| Status report | `/internal-comms` |
