# Development Workflows

## Language-Specific
- `/python-development` — Python 3.12+, Django, FastAPI, async
- `/javascript-typescript` — ES6+, Node.js, React, modern frameworks
- `/backend-development` — APIs, microservices, TDD patterns
- `/frontend-design` — Production-grade UI, modern design
- `/database-design` — Schema, migrations, query optimization

## AI/Integration
- `/llm-application-dev` — Prompt engineering, RAG, chatbots
- `/mcp-builder` — MCP servers for external API integration

## Creative
- `/algorithmic-art` — p5.js, seeded randomness, generative art

## Code Quality
- `/code-review` — Quality, security, performance audits
- `/code-refactoring` — Clean legacy code, reduce complexity
- `/code-documentation` — API docs, READMEs, inline comments

## Testing
- `/qa-regression` — Reusable test scenarios
- `/webapp-testing` — Playwright, screenshots, logs

## Workflow Selection

| Task | Workflow |
|------|----------|
| Python API | `/python-development` |
| React component | `/javascript-typescript` |
| UI design | `/frontend-design` |
| Database schema | `/database-design` |
| RAG system | `/llm-application-dev` |
| PR review | `/code-review` |
| Legacy cleanup | `/code-refactoring` |
| E2E tests | `/webapp-testing` |
