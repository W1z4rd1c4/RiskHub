# Plan 73-02: Config-Driven Permission Logic

> **Scope**: ~2 hours
> **Depends on**: Plan 73-01 (for fallback safety)

## Objective

Modify the permission system so that `can_resolve_approvals()` checks the `approval_scenarios.approver_roles` configuration from Risk Hub, creating a single source of truth.

---

## Context

Currently two disconnected systems:

| System | Controls |
|--------|----------|
| `role_permissions` table | `can_resolve_approvals()` behavior |
| `approval_scenarios.approver_roles` | Display in Risk Hub UI only |

This plan connects them so CRO can dynamically configure approvers via Risk Hub.

---

## Tasks

### 1. Create Async Permission Helper

**File**: `backend/app/core/permissions.py`

Add new async function:

```python
async def can_resolve_approvals_async(user: User, db: AsyncSession) -> bool:
    """
    Check if user can approve/reject requests.
    
    Uses approval_scenarios config for dynamic approver roles.
    Falls back to has_permission check if config unavailable.
    """
    if not is_privileged_user(user):
        return False
    
    # Get configured approver roles from any scenario
    from sqlalchemy import select
    from app.models.approval_scenario import ApprovalScenario
    
    result = await db.execute(select(ApprovalScenario).limit(1))
    scenario = result.scalar_one_or_none()
    
    if scenario:
        approver_roles = scenario.get_approver_roles()
        user_role = getattr(user.role, 'name', None)
        if user_role in approver_roles:
            return True
    
    # Fallback to traditional permission check
    return has_permission(user, "approvals", "write")
```

### 3. Add Caching (Optional Optimization)

To avoid DB lookup on every request:

```python
from functools import lru_cache
from datetime import datetime, timedelta

_approver_roles_cache: tuple[list[str], datetime] | None = None

async def get_approver_roles_cached(db: AsyncSession) -> list[str]:
    global _approver_roles_cache
    if _approver_roles_cache and datetime.now() - _approver_roles_cache[1] < timedelta(minutes=5):
        return _approver_roles_cache[0]
    
    # Fetch from DB
    result = await db.execute(select(ApprovalScenario).limit(1))
    scenario = result.scalar_one_or_none()
    roles = scenario.get_approver_roles() if scenario else ["risk_manager", "cro"]
    
    _approver_roles_cache = (roles, datetime.now())
    return roles
```
