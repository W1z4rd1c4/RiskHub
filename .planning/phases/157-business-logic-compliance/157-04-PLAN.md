<task id="3" title="Handle approver unavailability">
Add fallback for unavailable primary approver:

```python
async def get_fallback_approver(db, approval):
    """Get fallback approver if primary is unavailable."""
    if approval.primary_approver_id:
        primary = await db.get(User, approval.primary_approver_id)
        if primary and primary.is_active:
            return approval.primary_approver_id
    
    # Fallback to department head
    dept_id = await _get_approval_department_id(db, approval)
    if dept_id:
        dept = await db.get(Department, dept_id)
        if dept and dept.manager_id:
            return dept.manager_id
    
    # Last resort: any privileged user can approve
    return None  # Indicates escalation to privileged pool
```

</task>
