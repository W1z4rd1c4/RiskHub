# Plan: Performance & Load Testing (Lightweight)

## Objective
Verify the application performs well under realistic production load (~30 users max, ~10 active).

## Execution Context
- **Phase**: 17 Production Deployment
- **Plan**: 17-05
- **Depends on**: 17-04 (E2E Regression Suite)
- **Environment**: Docker (riskhub-backend, riskhub-db, riskhub-frontend)

## Approach
Lightweight load testing suitable for a small-scale enterprise deployment:
- **Parallel E2E tests** — Simulate concurrent user sessions with Playwright
- **API benchmarks** — Document endpoint response times with pytest-benchmark
- **Query analysis** — Enable PostgreSQL slow query logging

No Locust or stress testing needed for 30-user environment.
