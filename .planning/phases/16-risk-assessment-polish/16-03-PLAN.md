  - Compare against `previous_submission.answers` (from include_previous)
  - Count only keys in the active template question set

UI constraints:
- Use existing “glass card” styling; keep it compact (one small block, not a dashboard).
- If there is no submitted questionnaire yet: show a single line “No submitted assessment yet.”
