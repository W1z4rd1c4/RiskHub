<task type="auto">
  <name>Task 2: Add signature verification to webhook endpoint</name>
  <files>backend/app/api/v1/endpoints/directory.py</files>
  <action>
1. Import hmac, hashlib from stdlib
2. Add `x_webhook_signature: str | None = Header(None)` parameter to `receive_webhook`
3. Before processing, verify signature:
   - If `settings.webhook_secret` is set, require valid signature
   - Compute expected = hmac.new(secret.encode(), request_body, hashlib.sha256).hexdigest()
   - Compare with `sha256=` prefix stripped from header
   - Return 401 if mismatch or missing when secret is set
4. If secret is empty (dev mode), log warning and proceed
  </action>
  <verify>curl -X POST /api/v1/directory/webhook without signature returns 401 when WEBHOOK_SECRET is set</verify>
  <done>Webhook validates signature when secret configured, returns 401 on invalid/missing</done>
</task>
