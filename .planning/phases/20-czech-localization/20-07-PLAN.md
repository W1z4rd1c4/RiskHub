### Terminology Consistency

| English | Czech |

| Administrator | Administrátor |
| CRO | CRO (keep abbreviation) |
| Risk Manager | Risk manažer |
| Department Head | Vedoucí oddělení |
| Access Scope | Rozsah přístupu |

- Keep markdown formatting identical
- Keep ASCII diagrams (translate labels only)
- Keep code examples unchanged
- Keep URLs and file paths unchanged
- Translate table headers and content
- Translate alert blocks (NOTE, TIP, WARNING, etc.)
