# Plan: PDF/Excel Report Translation

## Objective
Internationalize PDF and Excel report generation to output documents in the user's selected language. Include Czech translations for all report headers, labels, and summary text.

## Execution Context
- **Phase**: 20 Czech Localization
- **Plan**: 20-06
- **Depends on**: Plan 20-05 (Backend API Messages)
