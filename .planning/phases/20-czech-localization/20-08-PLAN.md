## Translation Guidelines

### User-Friendly Terminology
Use accessible Czech language for end users:

| English | Czech (User-Friendly) |
|---------|----------------------|
| Risk | Riziko |
| Control | Kontrola |
| KRI | KRI / Klíčový indikátor rizika |
| Dashboard | Přehled |
| Workflow | Úkoly |
| Approve | Schválit |
| Submit | Odeslat |
| Archive | Archivovat |
| Filter | Filtrovat / Vybrat |
| Export | Exportovat / Stáhnout |

### Formatting Rules
- Keep markdown formatting identical
- Translate table headers and content
- Translate step-by-step instructions
- Keep screenshots references (add Czech screenshots in future)
- Translate alert blocks
- Keep code/path examples unchanged
