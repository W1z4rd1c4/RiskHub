# Plan 253-07: Session and Transport Consolidation

## Objective

Group auth/session behavior under `frontend/src/services/session/`, delete `bootstrapSessionCache.ts`, move timeout/network primitives into shared API transport, and remove raw `console.error` calls from touched auth/login flows.

## Steps

1. Create `frontend/src/services/session/` and migrate store, manager, bootstrap, logout suppression, refresh hint, and SSO helpers.
2. Delete the old top-level session/auth-cache modules.
3. Add shared request runtime under `frontend/src/services/api/requestRuntime.ts`.
4. Flatten `authApi` into named functions plus a thin exported service object.
5. Route touched auth/login/form flows through `frontend/src/services/logger.ts`.

- `cd frontend && npm run lint`
