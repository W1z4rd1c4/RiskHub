# Plan 180-09: E2E Test Data Verification

<objective>
Verify that the E2E test data seeded in Phase 179 enables previously-skipped E2E tests. Run targeted test suites and compare skip counts before/after seeding.
</objective>

---

## Context

### Phase 179 Dependency
This plan verifies the E2E test data created in Phase 179:
- 15 risks (9 cross-department ownership)
- 12 controls (14 risk links)
- 10 KRIs (4 cross-department reporters)
- 5 approval requests (PENDING + PENDING_PRIVILEGED)

### Target Test Suites
Based on Phase 179 analysis, these suites should benefit most:

| Suite | Skip Cause | Data Added |
|-------|------------|------------|
| `cross-department/*.spec.ts` | No cross-dept data | 9 cross-dept risks, 6 cross-dept controls |
| `approval-workflows/*.spec.ts` | No pending approvals | 5 approval requests |
| `entity-ownership/*.spec.ts` | No entities with owners | 15 owned risks, 12 owned controls |
| `permissions/*.spec.ts` | Empty lists | All E2E entities |

- Total tests
