# Plan 180-10: E2E Test Updates for Deterministic Scenarios

<objective>
Update E2E tests to use the deterministic E2E-prefixed data seeded in Phase 179, eliminating data-dependent skips and improving test reliability.
</objective>

---

## Context

### Problem
Current tests skip when they can't find data dynamically:
```typescript
if (rowCount === 0) {
    test.skip(true, 'No controls available');
}
```

### Solution
Use deterministic E2E entities that are always present:
- `E2E-XDEPT-*` for cross-department scenarios
- `E2E-UW-*`, `E2E-CLM-*` etc. for standard scenarios

### Task 2: Update Cross-Department Tests

#### [MODIFY] `frontend/e2e/cross-department/control-owner-access.spec.ts`

**Before:**
```typescript
const rowCount = await controlsPage.getRowCount();
if (rowCount === 0) {
    test.skip(true, 'No controls available');
}
```

**After:**
```typescript
import { E2E_CONTROLS } from '../fixtures/e2e-data';
import { loginAsDemoUser } from '../helpers/login';

// Login as the owner of the cross-dept control
await loginAsDemoUser(page, { email: E2E_CONTROLS.OPS_OWNS_IT.owner_email });

// Navigate directly to the specific control
await page.goto(`/controls?search=${encodeURIComponent('E2E-XDEPT-OPS-IT-CTRL')}`);
await waitForDataLoad(page);

// Now we KNOW the control exists - no skip needed
const row = page.locator('tr').filter({ hasText: 'E2E-XDEPT-OPS-IT-CTRL' });
await expect(row).toBeVisible();
```
