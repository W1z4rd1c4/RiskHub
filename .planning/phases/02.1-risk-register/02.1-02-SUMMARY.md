# Phase 2.1 - Plan 02 Summary: Control-Risk Linking and Forms

Implemented the Control-Risk relationship management UI and the Risk Create/Edit functional forms with multi-step wizard and scoring preview.

## Changes Made

### Component Layer
- **[NEW] [LinkManagementDialog.tsx](../../../frontend/src/components/LinkManagementDialog.tsx)**: Reusable dialog for managing bidirectional relationships between controls and risks. Features searchable results, effectiveness selection, notes, and management of existing links.
- **[NEW] [RiskForm.tsx](../../../frontend/src/components/RiskForm.tsx)**: Multi-step wizard (4 steps) for creating and editing risks. Includes:
    - Identity & Classification
    - Details & Ownership (using lookup services)
    - Visual Risk Assessment (live 5x5 matrix preview for Gross and Net risk)
    - KRI Definitions & Thresholds
- **[MODIFY] [ControlDetailPage.tsx](../../../frontend/src/pages/ControlDetailPage.tsx)**: Integrated `LinkManagementDialog` and added logic for handling risk linkages.
- **[MODIFY] [RiskDetailPage.tsx](../../../frontend/src/pages/RiskDetailPage.tsx)**: Integrated `LinkManagementDialog` and added logic for handling control linkages.

### Page Layer
- **[NEW] [RiskForms.tsx](../../../frontend/src/pages/RiskForms.tsx)**: Page wrappers `RiskNewPage` and `RiskEditPage`.
- **[MODIFY] [index.ts](../../../frontend/src/pages/index.ts)**: Exported new pages.
- **[MODIFY] [App.tsx](../../../frontend/src/App.tsx)**: Added routes for `/risks/new` and `/risks/:id/edit`.

### Service Layer
- **[MODIFY] [controlApi.ts](../../../frontend/src/services/controlApi.ts)**: Added `linkRisk` and `unlinkRisk` methods.
- **[NEW] [lookupApi.ts](../../../frontend/src/services/lookupApi.ts)**: Centralized lookup for users and departments (mocked for now).
