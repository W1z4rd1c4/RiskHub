<task type="auto">
  <name>Task 2: Implement derived-field recomputation and audit attribution for approval edits</name>
  <files>backend/app/services/approval_execution_service.py</files>
  <action>
Update `_apply_edit_risk_control`:

For Risk edits:
- After applying pending changes, if any of `gross_probability`, `gross_impact`, `net_probability`, `net_impact` changed, recompute:
  - `risk.gross_score = risk.gross_probability * risk.gross_impact`
  - `risk.net_score = risk.net_probability * risk.net_impact`
- Ensure the activity log change set includes score changes when recomputation changes stored values (so audit trail reflects persisted state).

For Control edits:
- If any field is applied, set `control.updated_by_id = current_user.id` (mirror other update paths).

Avoid:
- Writing audit/derived fields without ensuring they’re actually persisted.
- Logging score changes if scores didn’t change.

Step-by-step implementation guidance:
1. Track which fields were applied from `pending_changes` (you already have `applied_changes`).
2. For risks:
   - Compute booleans like `gross_inputs_changed = any(k in applied_changes for k in ("gross_probability","gross_impact"))` and similarly for net.
   - Recompute scores only if the corresponding inputs changed.
   - If recompute changes the stored score values, add entries to `applied_changes` for `gross_score` / `net_score` so the activity log matches DB state.
3. For controls:
   - If `applied_changes` is non-empty, set `control.updated_by_id = current_user.id`.
   - If you log changes, optionally include `"updated_by_id": {"old": ..., "new": current_user.id}` if it changed (keep it consistent with your audit patterns).

Fix example sketch (risk branch):
```py
gross_inputs_changed = any(k in applied_changes for k in ("gross_probability", "gross_impact"))
net_inputs_changed = any(k in applied_changes for k in ("net_probability", "net_impact"))

if gross_inputs_changed:
    old = risk.gross_score
    risk.gross_score = risk.gross_probability * risk.gross_impact
    if risk.gross_score != old:
        applied_changes["gross_score"] = {"old": old, "new": risk.gross_score}

if net_inputs_changed:
    old = risk.net_score
    risk.net_score = risk.net_probability * risk.net_impact
    if risk.net_score != old:
        applied_changes["net_score"] = {"old": old, "new": risk.net_score}
```

Fix example sketch (control branch):
```py
if applied_changes:
    control.updated_by_id = current_user.id
```
