# Plan: 06-04 Production Hardening

## Objective
Configure application security, performance, and monitoring for production environments.

## Execution Context
- **Backend**: FastAPI security middleware.
- **Environment**: Production variable overrides.

## Tasks
1. **Security Middleware**:
   - Configure `CORSMiddleware` with restricted origins.
   - Add `TrustedHostMiddleware`.
   - Implement security headers (HSTS, X-Content-Type-Options, etc.).
2. **Logging & Monitoring**:
   - Switch to JSON logging or structured logging format.
   - Configure log rotation and production log levels.
3. **Performance**:
   - Enable Gzip/Brotli compression in Nginx and Backend.
   - Configure DB connection pool for higher concurrency.

## Verification Plan
1. **Security Scan**: Use basic tools to check for presence of security headers.
2. **CORS Test**: Verify cross-origin requests from unauthorized domains are blocked.
3. **Log Check**: Verify production logs don't leak sensitive data (SQL queries, PII).
