# Plan: 06-02 UI Localization

## Objective
Fully localize the user interface into Czech (CZ) and implement a seamless language switcher.

## Execution Context
- **Frontend**: React components across all pages.
- **Translations**: Standardized keys for all UI labels.

## Tasks
1. **Dictionary Population**:
   - Extract all hardcoded strings from `Sidebar`, `Header`, and `Dashboard`.
   - Populate `cs.json` and `en.json` with comprehensive translations.
2. **Component Refactoring**:
   - Use `useTranslation()` hook in all major pages (`RisksPage`, `ControlsPage`, `DepartmentsPage`).
   - Replace static text with `t('key')` calls.
3. **Language Switcher**:
   - Add a premium language toggle (EN/CZ) to the `Header` component with micro-animations.
   - Persistence: Ensure the selection persists in `localStorage`.

## Verification Plan
1. **Visual Check**: Toggle between EN and CZ; verify all main navigation and headings update.
2. **Persistence**: Refresh page and verify the last selected language remains active.
3. **Edge Cases**: Ensure tooltips and chart labels (where possible) are also localized.
