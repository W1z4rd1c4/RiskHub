## Features Implemented

### Department List Page
- Displays all departments with real-time metrics
- Shows user count, risk count, control count per department
- Highlights high-risk departments
- Loading and error states
- Refresh button
- Click to navigate to detail page

### Department Detail Page
- **Header:** Department name, code, description, back button
- **Stats Row:** 4 key metrics with icons
- **Risks Tab:** Sortable table of department's risks with status, gross/net scores
- **Controls Tab:** Sortable table of department's controls with form, frequency, risk level
- **Activity Tab:** Recent control executions with results and executors
- All tables clickable to navigate to detail pages
