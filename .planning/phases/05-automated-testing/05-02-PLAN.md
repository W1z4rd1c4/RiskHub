# Plan 05-02: Frontend Testing with Vitest

## Objective
Establish frontend test coverage using Vitest and React Testing Library for component and integration testing.

## Scope

### In Scope
- Vitest + React Testing Library setup
- Component tests for key UI components
- API service mocking with MSW (Mock Service Worker)
- Page-level integration tests

### Out of Scope
- E2E browser testing (Playwright - Phase 6)
- Visual regression testing
- Performance testing

## Tasks

### Task 1: Install Testing Dependencies
Add to `package.json` devDependencies:
```json
"vitest": "^2.0.0",
"@testing-library/react": "^16.0.0",
"@testing-library/jest-dom": "^6.0.0",
"@testing-library/user-event": "^14.5.0",
"jsdom": "^24.0.0",
"msw": "^2.0.0"
```

### Task 2: Create Vitest Configuration
Create `frontend/vitest.config.ts`:
- jsdom environment
- Path aliases matching vite.config.ts
- Setup files for jest-dom matchers

Create `frontend/vitest.setup.ts`:
- Import jest-dom matchers
- MSW server setup

### Task 3: Create Test Utilities
Create `frontend/src/test/utils.tsx`:
- Custom render with providers (Router, Auth)
- Re-export testing-library methods

Create `frontend/src/test/mocks/handlers.ts`:
- MSW handlers for API endpoints
- Mock data factories

### Task 4: Create Component Tests
Create tests in `frontend/src/components/__tests__/`:
- [ ] `RiskScoreMatrix.test.tsx` - Matrix rendering, click handlers
- [ ] `ControlForm.test.tsx` - Form validation, submission
- [ ] `ExecutionLogModal.test.tsx` - Modal open/close, form logic
- [ ] `ExecutionHistory.test.tsx` - List rendering, expansion

### Task 5: Create Page Integration Tests
Create tests in `frontend/src/pages/__tests__/`:
- [ ] `DashboardPage.test.tsx` - Data loading, rendering widgets
- [ ] `ControlsPage.test.tsx` - List, filter, pagination
- [ ] `RisksPage.test.tsx` - List, filter, pagination
- [ ] `AuditTrailPage.test.tsx` - Execution list, filtering

### Task 6: Create API Service Tests
Create `frontend/src/services/__tests__/`:
- [ ] `controlApi.test.ts` - CRUD operations
- [ ] `riskApi.test.ts` - CRUD operations
- [ ] `executionApi.test.ts` - Log, list operations

### Task 7: Add Test Scripts
Update `package.json` scripts:
```json
"test": "vitest",
"test:run": "vitest run",
"test:coverage": "vitest run --coverage"
```

## Verification

```bash
cd frontend
npm run test:run
```

Expected: All tests pass with 0 failures.

## Success Criteria
- [ ] Vitest configured and running
- [ ] jest-dom matchers available
- [ ] MSW mocking API calls
- [ ] Key components have test coverage
- [ ] Page integration tests pass
- [ ] `npm run test` works correctly
