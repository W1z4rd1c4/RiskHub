# Plan 01-02: FastAPI Backend with SQLAlchemy and PostgreSQL

## Objective

Set up the Python FastAPI backend with SQLAlchemy ORM, PostgreSQL database connection, and base API structure. Create the foundation for all future API endpoints.

## Execution Context

**Tech Stack:**
- Python 3.11+
- FastAPI with async support
- SQLAlchemy 2.0 with async
- PostgreSQL database
- Pydantic for validation
- Alembic for migrations
- uvicorn for server

**Project Structure:**
```
backend/
├── app/
│   ├── api/
│   │   ├── v1/
│   │   │   ├── endpoints/
│   │   │   └── router.py
│   │   └── deps.py
│   ├── core/
│   │   ├── config.py
│   │   └── security.py
│   ├── db/
│   │   ├── base.py
│   │   └── session.py
│   ├── models/

│   └── main.py
├── alembic/
├── tests/

├── alembic.ini
└── .env
```

## Tasks

### Task 1: Initialize Python project
```bash
mkdir -p backend
cd backend
python -m venv venv
source venv/bin/activate
```

Create `requirements.txt`:
```
fastapi>=0.109.0
uvicorn[standard]>=0.27.0
sqlalchemy[asyncio]>=2.0.25
asyncpg>=0.29.0
pydantic>=2.5.0
pydantic-settings>=2.1.0
alembic>=1.13.0
python-multipart>=0.0.6
```

### Task 2: Create FastAPI application structure
- `app/main.py` — FastAPI app initialization with CORS
- `app/core/config.py` — Settings with Pydantic BaseSettings
- `app/api/v1/router.py` — API router aggregation

### Task 3: Configure SQLAlchemy with async PostgreSQL
- `app/db/session.py` — Async session factory
- `app/db/base.py` — SQLAlchemy Base class
- `app/api/deps.py` — Database dependency injection

### Task 4: Set up Alembic for migrations
```bash
alembic init -t async alembic
```

Configure `alembic.ini` and `alembic/env.py` for async SQLAlchemy.

### Task 5: Create base models
- `app/models/base.py` — Base model with id, created_at, updated_at
- `app/models/user.py` — User model (for role-based access)
- `app/models/department.py` — Department model

### Task 6: Create health check endpoint
```python
@router.get("/health")
async def health_check():
    return {"status": "healthy", "version": "1.0.0"}
```

### Task 7: Configure Docker for PostgreSQL (development)
Create `docker-compose.yml`:
```yaml
services:
  db:
    image: postgres:16
    environment:
      POSTGRES_USER: riskhub
      POSTGRES_PASSWORD: riskhub_dev
      POSTGRES_DB: riskhub
    ports:
      - "5432:5432"
    volumes:
      - postgres_data:/var/lib/postgresql/data
```

### Task 8: Create initial migration
```bash
alembic revision --autogenerate -m "initial"
alembic upgrade head
```

## Verification
