# Plan 01-01: React + Vite Frontend Scaffolding

## Objective

Set up the React frontend with Vite, Tailwind CSS, and shadcn/ui components. Create the base application structure with routing, layout components, and development environment.

## Execution Context

**Tech Stack:**
- React 18+ with TypeScript
- Vite for build tooling
- Tailwind CSS for styling
- shadcn/ui for component library
- React Router for navigation

**Project Structure:**
```
frontend/
├── src/
│   ├── components/     # Reusable UI components
│   ├── pages/          # Route pages
│   ├── layouts/        # Page layouts
│   ├── hooks/          # Custom React hooks
│   ├── lib/            # Utilities and helpers
│   ├── types/          # TypeScript types
│   └── App.tsx
├── public/
├── package.json
├── vite.config.ts
├── tailwind.config.js
└── tsconfig.json
```

## Tasks

### Task 1: Initialize Vite + React project
```bash
npm create vite@latest frontend -- --template react-ts
cd frontend
npm install
```

### Task 2: Install and configure Tailwind CSS
```bash
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

Configure `tailwind.config.js` and add Tailwind directives to CSS.

### Task 3: Install and configure shadcn/ui
```bash
npx shadcn@latest init
```

Select options:
- TypeScript: Yes
- Style: Default
- Base color: Slate
- CSS variables: Yes

Install core components:
```bash
npx shadcn@latest add button card input label select table tabs
```

### Task 4: Set up React Router
```bash
npm install react-router-dom
```

Create base routes:
- `/` — Dashboard (home)
- `/controls` — Control catalog
- `/departments` — Department management
- `/settings` — Settings

### Task 5: Create base layout components
- `MainLayout.tsx` — Sidebar + header + content area
- `Sidebar.tsx` — Navigation sidebar with menu items
- `Header.tsx` — Top header with user info placeholder
- `PageContainer.tsx` — Consistent page wrapper

### Task 6: Create placeholder pages
- `DashboardPage.tsx`
- `ControlsPage.tsx`
- `DepartmentsPage.tsx`
- `SettingsPage.tsx`

### Task 7: Configure development environment
- Set up `.env` for API URL
- Configure Vite proxy for API calls
- Add ESLint + Prettier configuration

## Verification

```bash
cd frontend
npm run dev
# Verify app loads at http://localhost:5173

# Verify Tailwind styles are applied
```
