# Plan 01-04: Premium Frontend Redesign

## Goal
Completely redesign the RiskHub frontend to deliver a "wow" experience. This includes a stunning hero landing page, a modern administrative dashboard with glassmorphism, and smooth micro-animations.

## Proposed Changes

### [Component] UI/UX Foundation
- **Design Tokens**: Implement a sophisticated HSL-based color palette (Deep Slates, Electric Indigos, Emerald accents).
- **Typography**: Switch to `Outfit` and `Inter` for a premium feel.
- **Animations**: Install `framer-motion` for transitions and hover effects.

### [Component] Landing Page (New)
- **Hero Section**: Large, bold typography with a mesh gradient background.
- **Mock Login**: A "Secure Enterprise Access" button that triggers a mock admin login.

### [Component] Dashboard Redesign
- **Sidebar**: Implement a semi-transparent, glassmorphism sidebar.
- **Cards**: Update statistic cards with better spacing, borders, and subtle hover lifts.
