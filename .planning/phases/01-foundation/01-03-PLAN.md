# Plan 01-03: Role-Based Access Structure (SII Roles)

## Objective

Implement the Solvency II compliant role-based access control (RBAC) structure with mocked authentication. Create the user and role models, permission system, and frontend auth context that will later integrate with Azure AD.

## Execution Context

**SII Role Hierarchy:**
```
Admin
├── Chief Risk Officer (CRO)
│   ├── Risk Manager
│   ├── Actuarial Function
│   └── Compliance Officer
├── Internal Audit (read-only)
├── Department Head
│   └── Control Owner
└── Viewer (read-only)
```

**Permission Model:**
- Resource-based permissions (controls, departments, reports)
- Action-based permissions (read, write, delete, approve)
- Hierarchical inheritance (CRO inherits Risk Manager permissions)

### Task 1: Create Role and Permission models (backend)

`app/models/role.py`:
```python
class Role(Base):
    __tablename__ = "roles"
    
    id: Mapped[int] = mapped_column(primary_key=True)
    name: Mapped[str] = mapped_column(String(50), unique=True)
    description: Mapped[str] = mapped_column(String(255))
    permissions: Mapped[list["Permission"]] = relationship()
```

`app/models/permission.py`:
```python
class Permission(Base):
    __tablename__ = "permissions"
    
    id: Mapped[int] = mapped_column(primary_key=True)
    resource: Mapped[str]  # controls, departments, reports, users
    action: Mapped[str]    # read, write, delete, approve
```

### Task 2: Extend User model with roles

```python
class User(Base):
    __tablename__ = "users"
    
    id: Mapped[int] = mapped_column(primary_key=True)
    email: Mapped[str] = mapped_column(unique=True)
    name: Mapped[str]
    role_id: Mapped[int] = mapped_column(ForeignKey("roles.id"))
    department_id: Mapped[int | None] = mapped_column(ForeignKey("departments.id"))
    is_active: Mapped[bool] = mapped_column(default=True)
    
    role: Mapped["Role"] = relationship()
    department: Mapped["Department"] = relationship()
```

### Task 5: Create mock authentication (backend)

`app/api/deps.py`:
```python
async def get_current_user(
    db: AsyncSession = Depends(get_db),
    user_id: int = Header(None, alias="X-Mock-User-Id")
) -> User:
    """Get current user - mocked for now, will integrate with Azure AD."""
    if user_id:
        return await db.get(User, user_id)
    # Default to admin for development
    return await db.scalar(select(User).where(User.role.has(name="admin")))
```
