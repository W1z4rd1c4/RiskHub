# Plan: Report Generation Backend

## Objective

Implement PDF and Excel export functionality for controls, risks, and dashboard data. Provide downloadable reports for auditors and executives.

---

## Scope

**In Scope:**
- PDF report generation (ReportLab)
- Excel export (openpyxl)
- Report endpoints for controls, risks, and summary
- Frontend download buttons

**Out of Scope:**
- Scheduled report generation (future)
- Email delivery (future)

---

**Files:**

**Implementation:**

### Task 3: Create Report Endpoints
Add API endpoints for downloading reports.

**Files:**
- `backend/app/api/v1/endpoints/reports.py` [NEW]
- `backend/app/api/v1/router.py` [MODIFY]

**Endpoints:**
| Method | Path | Description |
|--------|------|-------------|
| GET | `/api/v1/reports/controls/pdf` | Download controls as PDF |
| GET | `/api/v1/reports/controls/excel` | Download controls as Excel |
| GET | `/api/v1/reports/risks/pdf` | Download risks as PDF |
| GET | `/api/v1/reports/risks/excel` | Download risks as Excel |
| GET | `/api/v1/reports/summary/pdf` | Download dashboard summary as PDF |

**Query Parameters:**
- `department_id` (optional): Filter by department
- `status` (optional): Filter by status

**Response:**
- `Content-Type: application/pdf` or `application/vnd.openxmlformats-officedocument.spreadsheetml.sheet`
- `Content-Disposition: attachment; filename="controls-2025-12-25.pdf"`
