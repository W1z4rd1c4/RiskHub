# Plan: 06-02 Risk Appetite Frontend

## Objective
Create user-friendly KRI visualization with nice visuals, inline indicators, and edit capabilities.

## UI Flow
```
Risks List → [click row] → Risk Detail → [click KRI] → KRI Detail/Edit
     ↓                           ↓
  KRI count badge           KRI gauge cards
```

## Tasks

### 1. TypeScript Types
`frontend/src/types/kri.ts`:
```typescript
interface KeyRiskIndicator {
  id: number;
  risk_id: number;
  metric_name: string;
  current_value: number;
  lower_limit: number;
  upper_limit: number;
  unit: string;
  breach_status: 'above' | 'below' | 'within';
  last_updated: string;
}
```

### 2. API Service
`frontend/src/services/kriApi.ts`

### 3. Risk List Enhancement
Update `RisksPage.tsx`:
- Add column "KRIs" showing count badge
- Badge color: 🟢 green (all within), 🟡 yellow (near limit), 🔴 red (any breach)
- Example: "3 KRIs" or "⚠️ 2/3 breach"

### 4. KRI Gauge Card Component
`frontend/src/components/kri/KRIGaugeCard.tsx`:
- Premium glassmorphism card
- Bullet gauge: lower ← [value] → upper
- Color gradient based on breach status
- Click opens detail modal

### 5. Risk Detail KRI Section
Update `RiskDetailPage.tsx`:
- Add "Key Risk Indicators" tab/section
- Grid of `KRIGaugeCard` components
- Empty state: "No KRIs linked to this risk"

### 6. KRI Detail Modal
`frontend/src/components/kri/KRIDetailModal.tsx`:
- Full metric details
- Value history chart (if available)
- **Edit button** → opens inline edit form
- Fields: metric_name, current_value, lower_limit, upper_limit, unit

### 7. KRI Edit Form
`frontend/src/components/kri/KRIEditForm.tsx`:
- Inline editing within modal
- Save/Cancel buttons
- Validation: lower < upper, value is numeric

### 8. Dashboard Widget (Optional)
`KRIBreachWidget.tsx`:
- Count of active breaches
- Top 5 critical breaches with drill-down

## Visuals
- **Gauge**: Horizontal bullet chart (Recharts `BarChart` or custom SVG)
- **Colors**: Green (#22c55e), Yellow (#eab308), Red (#ef4444)
- **Animations**: Framer Motion on gauge fill
- **Style**: Glassmorphism cards matching existing UI
