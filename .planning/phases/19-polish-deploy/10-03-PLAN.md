# Plan: 06-03 Docker Scaffolding

## Objective
Containerize the application for production-ready on-premise deployment.

## Execution Context
- **Backend**: FastAPI running with Uvicorn.
- **Frontend**: Vite build served via Nginx.
- **Orchestration**: Docker Compose.

## Tasks
1. **Backend Dockerfile**:
   - Create multi-stage build using `python:3.13-slim`.
   - Optimize layers for dependency installation.
2. **Frontend Dockerfile**:
   - Build stage using `node:20-alpine`.
   - Server stage using `nginx:stable-alpine`.
   - Configure custom `nginx.conf` for SPA routing.
3. **Docker Compose**:
   - Create `docker-compose.yml` defining `backend`, `frontend`, and `postgres` services.
   - Implement healthchecks and environment variable passing.

## Verification Plan
1. **Build**: `docker-compose build`
2. **Launch**: `docker-compose up -d`
3. **Access**: Navigate to `localhost:80` and verify the app communicates with the containerized backend.
