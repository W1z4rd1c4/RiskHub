# Plan: 06-01 i18n Infrastructure

## Objective
Establish the foundational infrastructure for internationalization in both frontend and backend.

## Execution Context
- **Frontend**: React 19, `react-i18next`, `i18next`.
- **Backend**: FastAPI, custom middleware for `Accept-Language` header handling.

## Tasks
1. **Frontend Core Setup**:
   - Install `i18next`, `react-i18next`, and `i18next-browser-languagedetector`.
   - Create `frontend/src/i18n.ts` configuration.
   - Initialize `en.json` and `cz.json` dictionaries in `frontend/src/locales/`.
2. **Backend Infrastructure**:
   - Implement a context-aware translation helper or utility.
   - Add middleware to detect language from headers and store it in request-local context.
3. **Integration**:
   - Wrap `App.tsx` with `I18nextProvider`.
   - Verify that the application still loads correctly after initialization.

## Verification Plan
1. **Build**: `npm run build` in frontend.
2. **Check Log**: Verify no 404s for locale files.
3. **Smoke Test**: Manually toggle language in console state to see labels change.
