### 3. Technická architektura (Nová sekce)

**Obsah (z `.planning/codebase/`):**

**Tech Stack:**
- **Backend**: Python 3.13+, FastAPI, SQLAlchemy 2.0 (AsyncIO), PostgreSQL
- **Frontend**: React 19, TypeScript, Vite, Tailwind CSS, Shadcn/UI
- **Vizualizace**: Recharts, Framer Motion
- **Testování**: pytest, Vitest, Playwright

**Architektura:**
- Service-Oriented Architecture (Hybrid)
- RiskHub: Monolitický FastAPI backend + React SPA
- AD Emulator: Standalone mikroslužba pro emulaci Active Directory
- Komunikace: HTTP REST API s JWT autentizací

**Klíčové abstrakce:**
- Models → Schemas → Endpoints → Services → Integrations

## Technická implementace

### 1. Screenshoty jako Base64

Pro standalone HTML je nutné převést PNG soubory na Base64:
- Použít Python script pro konverzi
- Embedding přímo do `<img src="data:image/png;base64,...">`
- Alternativa: Použít relativní cesty pro demo

### 2. CSS pro nové sekce

- Screenshot galerie: CSS Grid 3 sloupce s modal overlay
- Architektura: Mermaid diagram nebo ASCII art
- Roadmapa: Timeline vizualizace

### 3. Jazykové úpravy

Veškerý nový text v profesionální češtině:
- Formální styl pro board prezentaci
