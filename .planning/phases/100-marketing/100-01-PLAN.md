# Phase 100-01: Marketing Presentation (Board Pitch)

Create a high-end, single-file HTML presentation in Czech designed to pitch the RiskHub platform to the board of directors. The presentation will showcase all implemented features with a premium aesthetic (glassmorphism, micro-animations) and persuasive flow.

## Goal
A stunning, standalone HTML file (`presentation.html`) that allows the board to navigate through slides presenting the app's capabilities, architecture, and value proposition in Czech.

## User Review Required
> [!IMPORTANT]
> The presentation will be in **Czech**. I will translate technical terms accurately within the context of OS 18 (Czech National Bank/insurance regulation).

## Tasks

### 1. Structure & Slide Content (Czech)
- [ ] **Slide 1: Intro** - RiskHub: Moderní platforma pro řízení rizik.
- [ ] **Slide 2: Problém & Řešení** - Consolidation of Excel files into a single source of truth.
- [ ] **Slide 3: Registr rizik** - Gross/Net scoring, OS 18 categories, heatmaps.
