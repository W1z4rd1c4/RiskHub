# Administrator Guide

## Risk Hub Configuration

Access **Risk Hub** from the sidebar to configure:
- Risk Types
- System Settings (thresholds, notifications)
- Approval Rules
- Roles
- Departments

## User Management

Access **Access Management** to:
- View all users
- Assign roles
- Set permissions
- Manage access scopes

## Admin Console

Access **Admin Console** to monitor:
- System Health
- Application Logs
- Active Sessions
