# Skills Resolution

Canonical skill-resolution contract for Codex in this repository.

Codex resolves skills from:

1. Repo: `./.codex/skills/`
2. User: `$CODEX_HOME/skills/` (usually `~/.codex/skills/`)

Usage rule:
- Use skills when the task clearly matches a skill workflow.

Operational note:
- Repo skills can be installed globally using `scripts/setup-dev.sh`.
