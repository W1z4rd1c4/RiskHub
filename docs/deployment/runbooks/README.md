# Deployment Runbooks

Operational runbooks for supported production deployment and day-2 recovery scenarios.

Use this directory for focused, task-specific deployment guidance that is subordinate to:

- `docs/deployment/README.md`
- `docs/deployment/reference.md`
- `docs/deployment/security-checklist.md`

Keep runbook commands aligned with the public `./scripts/install.sh production --target docker|linux` surface.
